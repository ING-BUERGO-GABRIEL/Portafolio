Proyectos Realizados

Migrador Rms

Rol: Desarrollo completo y soporte - Tecnologias: C#, .Net 4.8, Sql Server

El Migrador RMS representa una herramienta integral diseñada meticulosamente para desempeñar diversas funciones clave. Uno de sus roles principales es facilitar la migración masiva de datos, permitiendo la transferencia eficiente desde formatos como CSV y Excel hacia bases de datos.

Herramineta que destaca por su capacidad para generar plantillas personalizadas, simplificando así la extracción de datos del sistema en una variedad de formatos.

En el ámbito del mantenimiento de datos, el Migrador RMS asume la responsabilidad. No solo se limita a la migración inicial, sino que también abarca la gestión de la información. Esto incluye actividades como el mantenimiento para preservar la salud de la base de datos, la identificación y corrección de posibles errores, y la implementación de políticas efectivas de borrado de datos obsoletos.

Launcher Rms

Rol: Desarrollo completo y soporte - Tecnologias: C#, .Net 4.8, TCP/IP

Aplicación de doble función, actuando tanto como cliente como servidor. Diseñada para gestionar el versionamiento y las actualizaciones automáticas del sistema Ramses ERP. optimizando así la experiencia de usuario y la eficiencia operativa.

el Launcher RMS se encarga de sincronizar de manera eficiente los elementos esenciales para el sistema. Esto incluye la sincronización y descarga de DLL, informes, imágenes, y otros recursos cruciales para el funcionamiento óptimo de Ramses ERP.

Ramses

Rol: Desarrollo y soporte - Tecnologias: Pascal, .Net, Sql Server

Sistema de Planificación de Recursos Empresariales (ERP) integral. Su alcance abarca la gestión completa de procesos críticos, incluyendo pedidos, compras, inventario, contabilidad y logística.

Generación de informes detallados, integrados con todos los modulo. Esta característica permite a los usuarios acceder a análisis profundos de diversas áreas cubiertas por el sistema ERP, contribuyendo a una toma de decisiones estratégica informada.

SISEFT-VGDE

Rol: Desarrollo - Tecnologias: Php, Laravel 7, Bootstrap, MySql

SISEFT-VGDE emerge como un sistema integral dedicado al desarrollo frutícola, desempeñando un papel crucial en la gestión eficiente de la producción de durazno. Este sistema se presenta como un portal web especializado diseñado para el registro exhaustivo de la producción frutícola, ofreciendo una interfaz intuitiva que facilita el proceso para los productores del area de Vallegrande.

En su función principal, SISEFT-VGDE permite no solo el registro, sino también la visualización de datos de producción a través de gráficas y estadísticas detalladas, organizadas por zonas específicas. Esta capacidad brinda a los usuarios una visión holística y detallada de la producción, permitiendo la identificación de patrones y tendencias relevantes.

Ws-SingBooss

Rol: Desarrollo completo y soporte - Tecnologias: ASP.NET Core 7.0, Sql Server

API REST dedicada al Back End de la aplicación móvil Booss, su función esencial es la visualización de métricas estadísticas. Desarrollada con precisión, esta aplicación impulsa la experiencia del usuario al proporcionar gráficos detallados y comprensibles que ofrecen una representación visual efectiva de datos clave.

Implementación de web tokens JWT, una tecnología que no solo garantiza la seguridad de la información, sino que también habilita la gestión de diferentes niveles de roles y privilegios. Esta capacidad para adaptarse dinámicamente a las necesidades de diversos usuarios brinda flexibilidad y personalización, asegurando que cada persona tenga acceso a la información relevante según su nivel de autorización.
